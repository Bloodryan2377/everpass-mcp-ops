# personal/

Personal (non-EverPass-business) identity scaffold — placeholders only, no real content
yet. Mirrors the shape of `EVERPASS/SOUL.md` / `VOICE.md` / `AUDIENCE.md` / `DESIGN.md` (layer-1,
`NEEDS RYAN INPUT` markers instead of invented answers) but for me as an individual, not
for EverPass Media as a company.

## Why this is separate from EVERPASS/

`EVERPASS/` is company identity — brand, voice, audience, visual design for EverPass
Media. This directory is personal — my own background, voice, and preferences, for
anything that isn't speaking as the company. The privacy gate in `CLAUDE.md` (this
directory) keeps the two from bleeding into each other by accident.

## What's here

- `CLAUDE.md` — the privacy gate (HARD rule, auto-loaded under this subtree).
- `SOUL.md` — origin, contrarian beliefs, defining moments. Placeholder questions only.
- `VOICE.md` — personal tone, never-use list, off-limits topics. Placeholder questions only.
- `AUDIENCE.md` — who this serves outside EverPass. Placeholder questions only.
- `DESIGN.md` — personal visual style. Placeholder questions only.

All four content files are `NEEDS RYAN INPUT` until answered directly. See `CLAUDE.md`
rule 2.

A manual, optional, one-time action. This is a claude.ai account action, not a repo
action, so it has no effect here beyond this note existing somewhere discoverable:

1. Claude (claude.ai) -> Settings -> Capabilities -> Import memory.
2. Run the generated export prompt in ChatGPT or Gemini.
3. Paste the result back into Claude.

Nothing in this repo triggers, depends on, or is changed by this.
